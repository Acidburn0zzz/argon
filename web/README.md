


目录：
------
* comm:     公共目录，负责mobile和pybbs的公共handler，包括ajax和api的handler在此。其中comm/urls.py下有所有handler的总路由表
* mobile：  移动版（简化版）的主handler目录
* pybbs：   网页版的主handler目录
* static：  js/css/img 
* template：模版
* tornado： web框架
* server.py 服务器程序, python server.py即可


安装：
------
* 需事先安装tornado： easy_install tornado
